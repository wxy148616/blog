使用

免费功能

搜索Web Search

应用内置的有一些功能，可以自己双击修改
也可以点击下面的add custom search添加新的搜索

  • 配置
name 含义
Search URL 要搜索的url规则
Title 标题
Keyword 启动的该搜索的快捷键
Validation 点击后面的test可以进行测试自己的配置是不是对的
  • Search URL规则
    以百度为例子，随便搜索一个testtest，拿到的URL是这样的：https://www.baidu.com/s?ie=UTF-8&wd=testtest
    其中testtest是自己搜索的内容，把testtest替换成{query}
    替换以后得到: https://www.baidu.com/s?ie=UTF-8&wd={query}，粘贴到Search URL就可以了

也可以直接点击Search URL后面的Lookup自己解析规则，准确率如何就不知道了

  • 目前自己也就添加了这么几个search
Title URL Keyword(个人建议)
百度 https://www.baidu.com/s?ie=UTF-8&wd={query} bl
京东 https://search.jd.com/Search?keyword={query} jd
GitHub https://github.com/search?q={query} gh
stackoverflow http://www.stackoverflow.com/search?q={query} sof

本地搜索File Search

key 含义
find 直接查找
open 打开文件
in 查找文件内的内容
tags 根据标签查找

付费功能

Snippets

  • 最常用的一个功能之一，可以快捷的打出自己预存的数据，比如地址、邮箱之类的
name 含义
Name 名字
Keyword 查找改内容的快捷键
Collection 所在分组
Auto expansion allowed 允许自动补充
Snippet 预存的内容
  • Snippet也支持动态占位符
    比如:
    {time}：当前时间
    {date}：当前日期
    {datetime}：当前时间和日期

Clipboard History

  • 可以记录之前复制的内容
  • clear删除历史记录
