安装

安装sublime Text3

  • 安装就不多说了,有钱去官网,没钱去百度,实在找不到可以联系下面的邮箱

    安装中文包插件

  • 首先需要安装package control
  • 使用快捷键Shift+Ctrl+p,会出现一个输入框, 输入package control, 会出现一系列的选项,选择Package Control:Install Package,
  • 下边框会出现Loading [ ==]的字样, 随后会出现一个输入框,就可以自己安装插件了.
  • 输入中文包插件localized, 会出现一些选项,不出意外第一个就是的,名字为LocalizedMenu,点击安装就可以了.

配置Python3环境

新增编译环境

  • 点击Tools(工具) --> Build System(编译系统) --> New Build System(新建编译系统)
  • 会出现一个文件名为untitled.sublime-build,更改文件内容.
  • 保存文件, 更名为Python3.sublime-build.
  • 注意: 文件中的路径为自己的Python3路径

    使用新增的Python环境

  • 选择环境: Tools(工具) --> Build System(编译系统) --> Python3(就是自己刚刚新增的那个)
